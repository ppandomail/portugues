# Fonética

| Termina com ... | Pronúncia | Exemplo |
| -- | -- | -- |
| **-o/-os**     | /u/ /us/     | tudo /tudu/ |
| **-e/-es**     | /i/ /is/     | eles /elis/ |
| **-de**        | /dhy/        | tarde /tardhy |
| **-te**        | /tchi/       | noite /noitchy/ |
| **-em**        | /ein/        | bem /bein/ |
| **-ão/-am**    | /aun/        | união /uniaun/ |
| **-õe/-ões**   | /oin/ /oins/ | põe /poin/ |
| **-ãe/-ães**   | /ain/ /ains/ | mamãe /mamain/ |
| **-l+cons/-l** | /u/          | alma /auma/ |
