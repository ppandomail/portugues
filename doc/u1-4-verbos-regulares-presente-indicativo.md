# Verbos regulares - Presente do Indicativo

| Pronome | FAL**AR** | BEB**ER** | PART**IR** |
| -- | -- | -- | -- |
| eu    | fal**o**    | beb**o**    | part**o** |
| você  | fal**a**    | beb**e**    | part**e** |
| nós   | fal**amos** | beb**emos** | part**imos** |
| vocês | fal**am**   | beb**em**   | part**em** |
