# BUSCAR vs PROCURAR

|              |                                  | Exemplos                           |
| --           | --                               | --                                 |
| **BUSCAR**   | quando nós sabemos onde está     | Vou **buscar** as chaves de casa   |
|              | quando você marcou com alguém    | Você pode vir me **buscar**?       |
| **PROCURAR** | quando nós NÃO sabemos onde está | Vou **procurar** as chaves de casa |
