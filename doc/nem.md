# NEM

|           |                          | Exemplos                                              |
| --        | --                       | --                                                    |
| NEM       | mesmo que não            | Eu **nem** vou sair de casa hoje. Estou muito cansado |
| NEM       | não                      | Ele **nem** parece tão velho                          |
| QUE + NEM | comparações de igualdade | Ele é alto que **nem** o irmão                        |
| NEM       | expressões               | Hoje não vou sair **nem a pau**                       |
