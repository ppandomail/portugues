# Artigos indefinidos

* Usados diante de substantivo

  | Artigo indefinido | Exemplo 1 | Exemplo 2 |
  | -- | -- | -- |
  | **um**   | um animal       | um amigo |
  | **uma**  | uma bicicleta   | uma amiga |
  | **uns**  | uns animais     | uns homens |
  | **umas** | umas bicicletas | umas mulheres |

## Exemplos

### UM

* **um** animal
* **um** computador
* **um** filho
* **um** moço
* **um** morango
* **um** televisor
* **um** tom

### UMA

* **uma** bicicleta
* **uma** estrela
* **uma** garota
* **uma** letra musical
* **uma** lua
* **uma** piscina
* **uma** tangerina

### UNS

* **uns** abacaxis
* **uns** animais
* **uns** celulares
* **uns** óculos
* **uns** tons

### UMAS

* **umas** crianças
* **umas** folhas
* **umas** garrafas
* **umas** girafas
* **umas** laranjas
* **umas** letras musicais
* **umas** moças
