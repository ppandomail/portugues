# Pronunciação letra L

|  | Som | Exemplos |
| -- | -- | -- |
| **L + vogal**     | normal  | latir, louco, alejado, luz, livre               |
| **-L**            | u       | final, animal, sol, casal                       |
| **L + consoante** | u       | calma, asfalto, salvar, falta                   |
| **LH**            | ll      | olhar, alho, compartilhar, milho, filha, melhor |
