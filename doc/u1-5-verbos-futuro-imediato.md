# Verbos - Futuro imediato

* Formação: verbo IR no presente + verbo ação no infinitivo
* Não pôr "a" entre os verbos

  | Pronome | Verbo IR no presente + verbo ação no infinitivo |
  | -- | -- |
  | eu    | **vou falar**    |
  | você  | **vai beber**    |
  | nós   | **vamos partir** |
  | vocês | **vão dormir**   |

* Eu vou ir = Eu irei
