# Contrações de preposições

* **obrigatório**

## Com artigos definidos

| Preposição | o        | a        | os        | as        |
| --         | --       | --       | --        | --        |

| de         | **do**   | **da**   | **dos**   | **das**   |
| em         | **no**   | **na**   | **nos**   | **nas**   |
| a          | **ao**   | **à**    | **aos**   | **às**    |
| para       | para o   | para a   | para os   | para as   |
| por        | **pelo** | **pela** | **pelos** | **pelas** |
