# ASA vs ASSA

|      |       |      | Exemplos                                       |
| --   | --    | --   | --                                             |
| ASA  | /zzz/ | ala  | O Piolho tem **asas**                              |
| ASSA | /s/   | asar | A minha mãe sempre **assa** um frango nos domingos |

* O frango que a minha mãe **assa** no forno tem **asas**
* Eu vou **assar** uma **asinha** de frango
