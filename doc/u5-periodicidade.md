# Periodicidade

| % | Advérbio |
| -- | -- |
| 0   | **nunca / jamais** |
| 10  | **dificilmente / raramente** |
| 20  | **não muito / poucas vezes** |
| 30  | **de vez em quando** |
| 50  | **às vezes** |
| 75  | **com frequência / geralmente** |
| 100 | **sempre / constantemente** |

* **uma vez** ao ano
* **duas vezes** por mês
* **três vezes** por semana
* **...** ao dia

* **todo** sábado / domingo
* **todas** as terças / sextas
