# Signos

| Sinais de pontuação | Nome              |
| --                  | --                |
| **,**               | vírgula           |
| **;**               | ponto e vírgula   |
| **.**               | ponto             |
| **:**               | dois pontos       |
| **-**               | travessão         |
| **-**               | hífen ou tracinho |
| **""**              | aspas             |
| **()**              | parênteses        |
| **[]**              | colchetes         |
| **!**               | exclamação        |
| **?**               | interrogação      |
| **/**               | barra             |
| **...**             | reticências       |

---

| Sinais de matemáticas | Nome         | Operações     |
| --                    | --           | --            |
| **+**                 | mais         | soma / adição |
| **-**                 | menos        | subtração     |
| **x**                 | vezes        | multiplicação |
| **÷**                 | dividido por | divisão       |
| **=**                 | igualdade    | igual a       |

---

| Outras signais | Nome                    |
| --             | --                      |
| **@**          | arroba                  |
| **_**          | underline               |
| *              | asterisco               |
| **#**          | tralha ou jogo da velha |
