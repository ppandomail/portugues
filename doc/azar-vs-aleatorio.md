|           |         | Exemplos                               |
| --        | --      | --                                     |
| AZAR      | ! sorte | Ele tem muito **azar** (mala suerte)   |
| ALEATÓRIO | al azar | Vamos escolher um número **aleatório** |
| ACASO     | azar    | Isto não é fruto do **acaso**          |
