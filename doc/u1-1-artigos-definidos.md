# Artigos definidos

* Usados diante de substantivo

  | Artigo definido | Exemplo 1 | Exemplo 2 |
  | -- | -- | -- |
  | **o**  | o menino   | o professor |
  | **a**  | a menina   | a professora |
  | **os** | os meninos | os alunos |
  | **as** | as meninas | as alunas |

## Exemplos

### O

* **o** brinco
* **o** cantor
* **o** carro
* **o** copo
* **o** frio
* **o** gato
* **o** médico
* **o** Pablo

### A

* **a** autorização
* **a** cadeira
* **a** lata
* **a** Marcela
* **a** mulher
* **a** música
* **a** rua

### OS

* **os** cachorros
* **os** cantores
* **os** homens

* **os** mares
* **os** meninos
* **os** ursos

### AS

* **as** camisetas
* **as** canetas
* **as** cidades
* **as** mesas
* **as** músicas
* **as** praias
* **as** roupas
