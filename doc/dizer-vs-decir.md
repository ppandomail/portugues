# DIZER vs DECIR

|       | Espanhol | Exemplos                             |
| --    | --       | --                                   |
| DIZ   | dice     | Ele **diz** que não gosta de peixe   |
| DISSE | dijo     | Ele **disse** que não gosta de peixe |

## Conjugação verbo DIZER

| Pronome | Presente | Passado  |
| --      | --       | --       |
| Eu      | digo     | disse    |
| Você    | diz      | disse    |
| Nós     | dizemos  | dissemos |
| Vocês   | dizem    | disseram |
