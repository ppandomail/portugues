# Sotaque brasileiro

|             |             | Exemplos                  |
| --          | --          | --                        |
| **você**    | **"cê"**    | **"cê"** vai viajar?      |
| **vocês**   | **"cêis"**  | **"cêis"** vão viajar?    |
| **estou**   | **"tô"**    |                           |
| **estava**  | **"tava"**  |                           |
| **está**    | **"tá"**    |                           |
| **estamos** | **"tamos"** |                           |
| **-ando**   | **"-anu"**  | eu **"tô" "andanu"**      |
| **-endo**   | **"-enu"**  | eu **"tô" "comenu"**      |
| **-indo**   | **"-inu"**  | eu **"tô" "subinu"**      |
| **-R**      |             | eu vou **"fazê"** a janta |
| **-ES**     | **"-eis"**  | **"mêis"**                |
| **-EZ**     | **"-eiz"**  | **"fêis"**                |
| **também**  | **"tamém"** | Ele **"tamém"** viajou    |
