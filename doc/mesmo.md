# Usos de MESMO

* Enfatizar o grau de uma qualidade
  * Ela é bonita **mesmo**!
  * Eu acordei as %h da manhã. - Cedo **mesmo**!
  * Comprei 10 Kg de uva. Adoro uva! - Adora **mesmo**!

* Confirmar perguntas...
  * Este vestido é muito feio, você não acha? - É **mesmo**!
  * Essa viagem é muito cara, né? - É **mesmo**!
