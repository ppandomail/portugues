# Pronunciação letra Z (Zê)

|  | Som | Exemplos |
| -- | -- | -- |
| **Z-**                 | zzz | zumbido, zero, zorro                      |
| **vogal + Z + vogal**  | zzz | azar (mala suerte), dezena, azedo (ácido) |
| **-Z**                 | s   | dez, arroz, feliz, paz, aprendiz          |
|                        | sh  | talvez                                    |
