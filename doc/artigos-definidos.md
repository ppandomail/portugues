# Artigos definidos

* Usados diante de substantivo

| Artigo definido | Exemplo 1 | Exemplo 2 |
| -- | -- | -- |
| **O**  | o menino   | o professor |
| **A**  | a menina   | a professora |
| **OS** | os meninos | os alunos |
| **AS** | as meninas | as alunas |
